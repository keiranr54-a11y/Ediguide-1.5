# Ediguide — University Ranking Website (static)

This repo contains a static website (HTML/CSS/JS) ready for GitHub Pages.

## Files
- `index.html` — main page
- `css/style.css` — styles
- `js/main.js` — interactivity (includes optional Firebase support)
- `js/firebase-config.js` — template (create and paste your Firebase config to enable persistent comments)
- `data/rankings.json` — ranking data (100 entries)
- `methodology.html` — methodology page
- `privacy.html` — privacy policy (required for ad networks)
- `.gitignore`, `LICENSE`

## Run locally
Open `index.html` in a browser. For full fetch() access in some browsers, run a simple local server:
```bash
python -m http.server 8000
# then open http://localhost:8000
