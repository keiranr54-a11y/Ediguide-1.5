Methodology

How Ediguide Rankings are created

This page explains the methodology used to construct the Ediguide Global ranking displayed on this site.

Data sources

Rankings are derived from the uploaded Ediguide list and structured for display. In a production release, Ediguide would combine multiple indicators such as research output, citations, teaching reputation, international outlook, and employer reputation.

Scoring

Each university is assigned a numeric score per the source. Lower is better in the uploaded dataset (the file's original scoring system is preserved).

Limitations

  • This is a demo and should not be used as the sole basis for high-stakes decisions.
  • Data provided is as-is and may contain inconsistencies or country naming variations.

Updates

When new data is available, the JSON file in the data/ folder should be updated and the site re-pushed to GitHub.
